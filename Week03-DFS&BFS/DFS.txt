[ DFS 알고리즘 ]

- 깊이 우선 탐색 : 그래프에서 깊은 부분을 우선적으로 탐색하는 알고리즘
- 최대한 깊숙이 노드들을 방문한 후, 다시 돌아가 다른 경로를 탐색
- 스택 자료구조를 이용
- 재귀함수를 이용하면 더 간단하게 알고리즘을 구현 가능 > O(N) 으로 구현 가능

[ DFS 동작 과정 ]
1) 스택에 탐색 노드를 삽입 > 방문값을 true 로 변환
2) 스택의 최상단 노드에서 방문하지 않은 인접노드를 스택에 넣고, 방문값을 true 로 변환
3) 스택의 최상단 노드에서 방분하지 않은 인접노드가 없으면, 단순히 최상단 노드 꺼내기
4) 스택에 값이 없을 때까지 2, 3 번의 과정을 반복
